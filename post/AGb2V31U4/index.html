AndroidRSACrypto

Packages: com.jrsys.crypto.library

以Android Keystore 為基礎的 RSA 金鑰/憑證管理，以及 RSA 簽/驗章服務（支援 PKCS#1 及 PKCS#7 ）。

X509Certificate getCertificate() 從Android Keystore 內取得憑證。

void importCertificate(byte[] certData, String keyAlias) 匯入憑證至 Android Keystore。
void importCertificate(InputStream certInputStream) 匯入憑證至 Android Keystore。

ByteArrayOutputStream readP7SignedContent(byte[] p7Data) 讀取PKCS#7簽章內的 signed content。

byte[] signP7Data(byte[] toBeSign, String digestAlgorithm, String keyAlias) 使用預設金鑰簽署出 PKCS#7 格式的數位簽章。
